真的是一道十分困难的题目！

（但是和双端队列一点关系也没有）

第一步想到的果然还是模拟与构造。

但是按照样例的提示来做的话十分不方便，因为存在一个输入顺序的问题。

反向思考来试试？

因为最后合成的大队列里数字是有序排列的。

我们不妨先按照数字的大小排序，并记录读入的顺序。

A[ ]来存储数字的大小 ， B[ ]来存储读入的顺序

用样例来说明的话，就是：

A[] = {0 , 3 , 3 , 6 , 6 , 9}
B[] = {3 , 1 , 6 , 2 , 5 , 4}

那么最终合成的大队列应该是A[ ]的形式的。

而且达达同学用的是双端队列，不放我们来模拟一下她的做法吧！

顺序读入 操作 队列
3 存入队列A A[ ] = {3} ; B[ ] = {}
6 存入队列B A[ ] = {3} ; B[ ] = {6}
0 插入队列A队头 A[ ] = {0,3} ; B[ ] = {6}
9 插入队列B队尾 A[ ] = {0,3} ; B[ ] = {6,9}
6 插入队列B队头 A[ ] = {0,3} ; B[ ] = {6,6,9}
3 插入队列A队尾 A[ ] = {0,3,3} ; B[ ] = {6,6,9}

合成的大队列：

C[] = {0 , 3 , 3 , 6 , 6 , 9}

我们观察后可以发现，与按照数字大小排序以后的读入顺序有关，例如，对于A[ ] = {0，3，3}来说，它的顺序B[ ] = {3，1，6}是一个单谷函数。

在谷底的时候读入，而后在递减的那一段插在队头，在递增的那一段插在队尾。

别走！！！真正的难点才刚来！！！

如果两个数一样，那么他们的读入顺序该如何排序呢？

可想而知的是应该尽可能的形成递减或者递增。

例如：

A[] = {6 , 6 , 9}
B[] = {2 , 5 , 4}

就是明显不对的，应该改为：

B[] = {5 , 2 , 4}

还可以看出一件值得注意的事情：数字相同的话，读入顺序的排序是可以随意交换的。

对于两个相同的数字，只要合并成一个，那就可以了是吧？

#pragma GCC optimize(3,"Ofast","inline")//O3只是皮一下（诶嘿~）
#include
#define PII pair
using namespace std ;
const int N = 200005 ;

inline int read() 
{
    int x = 0 , f = 1 ; char ch = getchar() ;
    while(!isdigit(ch)) {if(ch == '-') f = -1 ; ch = getchar() ;}
    while(isdigit(ch)) {x = (x << 1) + (x << 3) + ch-48 ; ch = getchar() ;}
    return x * f ;
}

inline void write(int x)
{
    if(x < 0) x = ~x + 1 , putchar('-') ;
    if(x > 9) write(x / 10) ;
    putchar(x % 10 + '0') ;
}

int n , cnt , ans , maxx[N] , minx[N] , h = INT_MAX ;
bool f = true ;// f == true 则代表当前为递减
PII p[N] ;
int a[N] ;

bool cmp(PII a , PII b)
{
    if(a.first == b.first) return a.second < b.second ;
    return a.first < b.first ;
}

signed main()
{
    n = read() ;
    for(int i = 1 ; i <= n ; ++ i)
        p[i].first = read() , p[i].second = i ;//合并相同的数的操作
    sort(p + 1 , p + 1 + n) ;
    for(int i = 1 ; i <= n ; ++ i)
        if(p[i].first != p[i - 1].first || i == 1)
        {
            maxx[cnt] = p[i - 1].second ;
            minx[++ cnt] = p[i].second ;
        }
    maxx[cnt] = p[n].second ;
    
    for(int i = 1 ; i <= cnt ; ++ i)
    {
        if(f)
        {
            if(h < minx[i])
                h = maxx[i] ;
            else
                h = minx[i] , ans ++ , f = false ;//说明现在到达谷顶了，需要再来一个队列
        }
        else
        {
            if(h > maxx[i])
                h = minx[i] ;
            else
                h = maxx[i] , f = true ;//到达谷底，更新f
        }
    }
    write(ans) ;
        
    return 0 ;
}
